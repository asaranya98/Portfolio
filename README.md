# 🚀 Saranya A — Data Analyst Portfolio

A modern, high-performance web portfolio tailored for a Data Analyst, designed to beautifully showcase technical skills, data-driven projects, professional experience, and certifications. Built with a customized **High-Tech Anti-Gravity UI** design language featuring glassmorphism, fluid animations, and a cinematic modern aesthetic.

## 🌟 Overview

As an entry-level Data Analyst passionate about turning messy datasets into crystal-clear insights using tools like Power BI, SQL, and Python, this portfolio serves as my centralized personal hub. It allows visitors and recruiters to:
- Explore my featured EDA projects and Dashboards.
- Browse my comprehensive technical stack and tools.
- Follow my progression through internships and academic timelines.
- Easily download my resume and certificates.

## 🛠️ Built With

- [React.js](https://reactjs.org/) & [Vite](https://vitejs.dev/) — Lightning-fast component framework and build tool.
- [Tailwind CSS](https://tailwindcss.com/) — Scalable, utility-first styling system.
- [Framer Motion](https://www.framer.com/motion/) — Powerful cinematic animations and spring physics.
- [Lucide React](https://lucide.dev/) — Clean, crisp vector icons.
- **React Router** — Seamless client-side navigation.

## 🪄 Key Features

- **Anti-Gravity UI:** Floating elements, ambient moving mesh gradients, and interactive 3D tilt cards.
- **Dynamic Micro-Interactions:** Infinite scrolling skill tickers, real-time stat counting animations, and dynamic routing.
- **SEO Optimized:** Ready for web crawlers with rich meta descriptions, Open Graph data, and targeted search keywords.
- **Fully Responsive:** Carefully sculpted to look premium on both mobile devices and ultrawide desktops.

## 🚀 Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/asaranya98/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Navigate to the preview:**
   Open your browser to `http://localhost:5173` (or the port specified in your terminal) to explore the portfolio.
